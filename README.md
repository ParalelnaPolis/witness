# Witness

## Insall

```bash
npm install
```

## Run verification

```bash
node index.js --page='https://paralelnapolis.sk/' --email='info@paralelnapolis.sk' --witness='Mgr. Pavol Trávnik' --organisation='PARALELNÁ POLIS, občianske združenie IČO: 51221900, v.z. štatutárny orgán Mgr. Pavol Trávnik'
```



### Examples
